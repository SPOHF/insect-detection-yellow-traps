# SWD Monitoring Frontend

React + Vite UI for:
- Login/register
- Map-based field drawing and trap placement
- Click trap marker and upload date-range trap images to that exact location
- Viewing upload/detection results
- Admin overview

## Run

```bash
cd apps/frontend
cp .env.example .env
npm install
npm run dev
```

Frontend: `http://localhost:5173`
