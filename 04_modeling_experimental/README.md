# 04 Modeling Experimental

All experimental modeling/training work is located here.

## Contents

- `src/` -> modeling code and CV core modules
- `configs/` -> config files for CLI/training/evaluation
- `data/` -> experiment datasets
- `runs/` -> generated run outputs
- `scripts/` -> helper scripts
- `tests/` -> tests for modeling and shared modules
- `weights/` -> weights/artifacts

## CLI usage

```bash
PYTHONPATH=04_modeling_experimental/src .venv/bin/python -m cli list-approaches
```
